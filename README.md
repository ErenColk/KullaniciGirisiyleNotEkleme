# Not Ekleme

Not Ekleme, .NET Core ve C# ile geliştirilmiş bir masaüstü uygulamasıdır. Kullanıcılar not ekleyebilir ve silebilirler. Birden fazla üyelik oluşturabilirler. Üye olma sırasında güvenliği sağlamak için şifre zorluk seviyeleri bulunmaktadır. 'Beni Hatırla' özelliği sayesinde kullanıcılar uygulamayı açtıklarında şifrelerini tekrar girmek zorunda kalmazlar.
